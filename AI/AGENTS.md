# STQ Agent Entry Point

This repository uses a lightweight skill harness system.

## Start here

Read:
- CLAUDE.md
- AI/AGENT_GUIDE.md

Then load relevant skills from:
- skills/*/SKILL.md

## Rules

- Keep PRs small
- Avoid unrelated refactors
- Preserve runtime/editor separation
- Prefer deterministic behavior
- Preserve backwards compatibility when possible

## Workflow

1. Read relevant skills
2. Analyze affected files
3. Propose minimal PR scope
4. Implement conservatively
5. Run verification
