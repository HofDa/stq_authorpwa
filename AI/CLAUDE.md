# STQ Agent Rules

## Core principles

- Preserve modular architecture
- Avoid large refactors unless explicitly requested
- Prefer conservative PR-sized changes
- Mobile-first UX
- Reuse existing design tokens
- Keep authoring UI touch-friendly
- Use progressive disclosure for complex UX
- Prefer composition over monolithic components

## Workflow

- Read relevant SKILL.md files before implementation
- Suggest PR breakdowns for large features
- Keep diffs minimal and reviewable
- Verify TypeScript types before completion

## Environment

- Node 22
- pnpm
- Vite
- React
- Tailwind v4
